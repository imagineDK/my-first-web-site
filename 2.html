CSS

Cascading Style Sheets (CSS) is a style sheet language used for describing the presentation of a document written in a markup language such as HTML or XML (including XML dialects such as SVG, MathML or XHTML).[1] CSS is a cornerstone technology of the World Wide Web, alongside HTML and JavaScript.[2]

CSS is designed to enable the separation of content and presentation, including layout, colors, and fonts.[3] This separation can improve content accessibility; provide more flexibility and control in the specification of presentation characteristics; enable multiple web pages to share formatting by specifying the relevant CSS in a separate .css file, which reduces complexity and repetition in the structural content; and enable the .css file to be cached to improve the page load speed between the pages that share the file and its formatting.

html내에서 css를 사용하는 두가지 방법

1. style이라는 태그를 통해 묶어주고 이 태그 안쪽에 css문법을 사용, 셀렉터 {} 중괄호를 사용하며 선언이라고 함
2. html 태그 안에 style="~~"이라는 속성 구문을 넣고 css문법을 사용


Style

color: 폰트의 컬러 의미. (ex. a {color:red;} - 링크되는 폰트는 모두 레드로 적용)
text-decoration: 텍스트의 장식 제어. (ex. 언더라인 유무 등)
font-size: 폰트의 크기. 픽셀, 센치, 퍼센트, 스몰, 라지 등 다양한 방식으로 정의 가능
text-align: 텍스트의 정렬 방식. center를 넣으면 중앙정렬 등
border-width: 아웃라인의 두께
border-color: 아웃라인 컬러
border-style: 아웃라인 스타일
border-bottom을 입력하면 보더의 밑줄만 사용
중복되는 스타일을 한번에 쓸 수 있다. ex. border: 5px solid red;
display: inline 을 하면 블록레벨 엘리먼트가 인라인 엘리먼트로 변한다
display: block 을 하면 인라인 엘리먼트가 블록레벨 엘리먼트로 변한다
display: none 을 하면 해당 부분이 사라진다
padding: 패딩 여백 값을 설정한다
margin: 엘리먼트 간 간격을 설정한다
display: grid 를 그리드를 제작한다

ex. .saw { color:gray }
id의 경우 #을 앞에 붙여준다. ex. #active {~~~}

블록레벨 엘리먼트: 엘리먼트를 포함한 화면 전체범위를 사용한다
ex. h1태그를 실행하면 자동줄바뀜이 되어있는데 이는 이 태그가 화면을 다 차지하고 있어서이다
인라인 엘리먼트: 엘리먼트 자체 크기만큼을 범위로 사용한다.
ex. a태그를 실행하면 줄바뀜이 이뤄지지않는데 이는 이태그가 자기크기만큼만 범위로 사용하기 때문이다


Grid

display: grid;
grid-template-columns: 150px 1fr; 을 입력하면 첫 열은 가로 150픽셀로 고정된 박스를 만들고 두번째 열은 화면을 꽉 채운다.
grid-template-columns: 1fr; 1fr; 을 입력하면 첫 열, 둘째 열이 1:1 비율로 화면을 꽉 채운다
grid-template-columns: 2fr; 1fr; 을 입력하면 첫 열, 둘째 열이 2:1 비율로 화면을 꽉 채운다
즉, fr은 전체화면을 기준으로 화면을 나누는 비율을 조정하는 단위이다
