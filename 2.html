Cascading Style Sheets (CSS)

안녕하세요. 저는 디자이너 디케이입니다.

CSS를 공부하는 공간입니다.




Tag 종류

h1~h6: 제목 크기
strong: 볼드
p: 단락구분, css연동시 마진값 정확히 입력 가능
br: 줄바꿈
u: 밑줄
img: 이미지넣기
li: 리스트, 항목의 리스트화, 앞에 점이 생김-항상 자식
ul: 언오더드리스트, li의 부모. 같은계열 항목끼리 감싸주면 구분됨, 비주얼적 변화는 없음
ol: 오더드리스트, li의 부모. 리스트 앞에 숫자를 자동으로 삽입
title: 웹페이지 제목
meta charset="utf-8": 문자 저장방식. 이걸 해줘야 문자 깨짐이 없다
head: 타이틀 및 헤드관련 정보를 묶는 태그
body: 본문을 묶는 태그
html: 프로젝트 전체를 묶는 태그
!DOCTYPE html: 제일 위에 관용적으로 적는다. html로 만든 코드라는 것을 표현
a: 링크 태그. 앵커의 약자


Attribute(속성) 종류

width: 가로사이즈
src: 소스 가져올 때
href: 하이퍼텍스트 참조(레퍼런스) 라는 뜻. a태그 뒤에 링크를 넣을 때 사용
target="blank"를 링크 뒤에 달아주면 새창으로 열린다. 없으면 페이지변경으로 열림
title="~~~"을 링크 뒤에 달아주면 마우스 호버시 간단한 타이틀명이 보인다
