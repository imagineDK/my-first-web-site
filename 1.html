디케이의 HTML 자습서

Hypertext Markup Language (HTML)

안녕하세요. 저는 디자이너 디케이입니다.

html을 공부하는 공간입니다.




Tag 종류

h1~h6: 제목 크기
strong: 볼드
p: 단락구분, css연동시 마진값 정확히 입력 가능
br: 줄바꿈
u: 밑줄
img: 이미지넣기
li: 리스트, 항목의 리스트화, 앞에 점이 생김-항상 자식
ul: 언오더드리스트, li의 부모. 같은계열 항목끼리 감싸주면 구분됨, 비주얼적 변화는 없음
ol: 오더드리스트, li의 부모. 리스트 앞에 숫자를 자동으로 삽입
title: 웹페이지 제목
meta charset="utf-8": 문자 저장방식. 이걸 해줘야 문자 깨짐이 없다
head: 타이틀 및 헤드관련 정보를 묶는 태그
body: 본문을 묶는 태그
html: 프로젝트 전체를 묶는 태그
!DOCTYPE html: 제일 위에 관용적으로 적는다. html로 만든 코드라는 것을 표현
a: 링크 태그. 앵커의 약자
iframe: 동영상 링크
!-- ~ -- : 구간 구문 무시
style: 이 태그 안쪽은 html내에서 css의 문법을 사용하겠다는 뜻.
- 특정 태그에 공통적으로 스타일을 적용할 때 용이
- 셀렉터라는 이 중괄호안에 스타일을 '선언'한다 { }

div: 아무 의미없이 사용하는 태그(주로 디자인 용도로 사용) division의 약자. 기본상태: 블록레벨 엘리먼트
span: div와 같은 목적의 태그. 기본상태: 인라인 엘리먼트
link: 페이지마다 중복되는 스타일 구문을 css파일로 만들어서 링크로 불러올 수 있음. 효율성 증대


Attribute(속성) 종류

width: 가로사이즈
src: 소스 가져올 때
href: 하이퍼텍스트 참조(레퍼런스) 라는 뜻. a태그 뒤에 링크를 넣을 때 사용
target="blank"를 링크 뒤에 달아주면 새창으로 열린다. 없으면 페이지변경으로 열림
title="~~~"을 링크 뒤에 달아주면 마우스 호버시 간단한 타이틀명이 보인다
style="~~~"라는 속성을 통해 html태그 안에서 css문법을 '선언' 가능하게 해준다

class & id 등은 주로 css 셀렉터를 통해 제어

class="~~~"를 하면 같이 지정된 클래스 끼리 그룹으로 묶는 효과를 준다
ex. class="saw" : 이미 본 페이지들 클래스화(그룹화)
"active" : 현재 활성화 된 페이지 클래스화
그리고 클래스 하나에 여러 속성 넣을 수 있다. 띄어쓰기로 분간. saw active ...

id="~~~": 클래스보다 좀 더 구체적인 지정용으로 사용. ex. id="active"
아이디로 지정하는 값은 전체 페이지에서 오직 한번만 사용되어야 한다. 클래스보다 우선순위가 상위이다

우선순위 : id 셀렉터 > class 셀렉터 > 일반 셀렉터
만약 동등한 것들 끼리 붙으면 나중에 등장한 코드가 우선순위 우위
